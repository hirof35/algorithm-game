# High & Low アルゴリズムゲーム (Web GUI版)

Go言語の標準ライブラリ（`net/http`）のみを使用して構築された、環境依存のないシンプルな数字当てゲームです。
プレイヤーが数字を当てる「ユーザーモード」と、コンピュータが二分探索（Binary Search）アルゴリズムを用いて自動で数字を絞り込む「AIモード」の2つを同時に楽しむことができます。

CGO（GCCコンパイラ）を一切使用しない設計になっているため、Windows、Mac、LinuxなどのOSやコンパイラのバージョンに関わらず、Go環境さえあればどこでも1発で起動します。

## 機能特徴

*   **2つのゲームモードを同時並行プレイ**
    *   **プレイヤーモード:** 1〜100の中からランダムに選ばれた数字を、ヒント（High/Low）を頼りにプレイヤーが当てるモード。
    *   **コンピュータモード:** プレイヤーが心に決めた数字を、コンピュータが「二分探索（Binary Search）」を使って最適解（最大7回以内）で当てにいくモード。
*   **環境依存ゼロのWeb GUI**
    *   面倒なGUIツールキットのインストールや、C言語コンパイラ（MinGW/GCC）のセットアップは完全に不要です。
    *   起動するとローカルPC上に超軽量なWebサーバーが立ち上がり、使い慣れたWebブラウザ上で操作できます。
*   **不正防止（矛盾検知）ロジック**
    *   AIモードにて、プレイヤーが嘘のヒントを入力して探索範囲が逆転した場合（`low > high`）、矛盾を検知して警告を出します。

## 動作環境

*   Go 1.16 以上 (CGO不要 / Pure Go)
*   モダンなWebブラウザ（Chrome, Edge, Safari, Firefoxなど）

## セットアップと起動方法

1.  **プロジェクトディレクトリの初期化（未実施の場合）**
```bash
    go mod init highlow-game
    ```

2.  **プログラムの起動**
    ターミナル（またはPowerShell）で以下のコマンドを実行します。
```bash
    go run main.go
    ```

3.  **ゲーム画面へのアクセス**
    起動後、ブラウザを開いて次のURLにアクセスしてください。
    *   [http://localhost:8080](http://localhost:8080)

## アルゴリズムについて（二分探索）

本ゲームのコンピュータモードでは、最悪時間計算量 $O(\log n)$ の**二分探索（Binary Search）**を採用しています。
1から100までの整数（$n = 100$）を探索空間とする場合、毎回探索範囲を半分に絞り込むため、数式は以下のようになります。

$$\log_2 100 \approx 6.64$$

これにより、プレイヤーがいかなる数字を選んでも、コンピュータは**最大7回以内**の試行で必ず正解に到達します（矛盾したヒントを出さない限り）。
